# Postman Collection Test1

Welcome to the Postman Collection Test1 repository!

## Description

This repository contains a collection of Postman API requests and tests for testing and documenting API endpoints.

## Getting Started

### Prerequisites

- [Postman](https://www.postman.com/downloads/) installed on your machine
- Git for cloning this repository

### Installation

1. Clone this repository:
```bash
git clone https://github.com/ghonu22/Postman_Collection_Test1.git
Open Postman and import the collection:
Click on Import button
Select the collection file from the cloned repository
The collection will be loaded into your Postman workspace
Usage
Once the collection is imported in Postman:

Navigate to the collection in your Postman workspace
Select the requests you want to test
Configure any required environment variables
Click Send to execute the API requests
Features
Comprehensive API request samples
Pre-configured request templates
Test scripts for automated validation
Environment variable support
